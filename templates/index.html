About

This a movie search web application named MSA utilizing HTML, CSS, JavaScript, Flask, and Python. By integrating the TMDb API, users can seamlessly explore a vast collection of movies, accessing key details and ratings. This project showcases my proficiency in frontend design, backend development, and API integration. MSA demonstrates my ability to create a user-friendly and interactive platform for movie enthusiasts.
